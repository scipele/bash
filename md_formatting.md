# Markdown

## Block Quotes
> single indented block quoted line
>> double indented block quoted line

## Lists:
Unordered List:
- Item 1
- Item 2
  - Nested Item A
  - Nested Item B

Ordered List:
1. First item
2. Second item
3. Third item

Task List:
- [x] Completed task
- [ ] Incomplete task

## Code Blocks:
- Inline Code: Wrap text in single backticks `code` to format short code snippets in a sentence.

<code class="language-cpp">std::cout << "Hello World\n"</code>

```python
# Fenced code block python example
def hello_world():
    print("Hello, World!")
```

```cpp
// C++ Example
std::cout << "hello world\n"
```

## Tables:
| Header 1     | Header 2       | Header 3      |
| :---         | :---:          | ---:          |
| Left-aligned | Center-aligned | Right-aligned |
| Row 2 Col 1  | Row 2 Col 2    | Row 2 Col 3   |

- Footnotes: Use [^1] to create a reference link to a footnote located at the bottom of the page. [1, 2, 3, 4]

## Text Emphasis:
- **This text is bold**
- __This text is also bold__
- *This text is italic*
- _This text is also italic_
- ***This text is bold and italic***
- ___This text is bold and italic___
- ~~This text has a strikethrough~~
- ==This text is highlighted==
- Highlight: Wrap text in double equal signs ==text== (Supported by platforms like Obsidian). [1, 2, 3, 4, 5]

## Structure & Headers:
- Headings: Use 1 to 6 hash symbols (# to ######) followed by a space at the start of a line to create H1 through H6 headers.
# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6
- Horizontal Rules: Type three or more dashes ---, asterisks ***, or underscores ___ on a blank line to create a dividing line.
